## Quickstart

- ```git clone https://github.com/archius11/med_one_api.git```

- ```cd med_one_api```

- ```cp example.env .env```

- Fill blank fields in .env

- ```docker-compose build```

- ```docker-compose up -d```

Working backend you can find at http://172.21.35.119:4567/docs
